    - fix: 버그 수정
    - docs: 문서 수정
    - style: 코드 포맷팅, 오타 수정, 주석 수정 및 삭제 등
    - refactor: 코드 리팩터링
    - chore: 빌드 및 패키지 수정 및 삭제
    - merge: 브랜치를 머지
    - ci: CI 관련 설정 수정
    - test: 테스트 코드 추가/수정
    - release: 버전 릴리즈
